FAQ

Once MR is installed and opened the following may happen:

  1. MR is asking for permissions: Depending on the device and the Android version, MR might ask for permissions to record audio, record videos, save files in your device and/or permission to allow display over other apps. Those permissions are necessary for MR to work and the user has to grant MR those permisisons in order to be able to record media.
  2. MR shows the message "MR does not have the required permissions to record": In this case MR needs that the user grants all the permissions that MR needs to be able to record media.
  3. MR is opened and no message is shown: In this case MR is ready to be used.

When MR is ready to use, the user have the following options:

  1. Go to "Settings" at the three vertical dots on the top right corner: Here you can set the general settings of MR and read information about MR. In general settings, the user has the following options:
    • Select the recording camera from the list of cameras available in the device.
    • Set the maximum recording lenght for each recorded file.
    • Enable/disable vibration notifications when MR has started/stopped recording.
    • Set the recording mode between video files or audio files.
    • Enable/disable the option of autostart. When Autostart is active MR will be automatically active everytime is opened.
    • Enable/disable the option to hide files. When Hide MR files is active all the recorded files are hidden from the device's gallery.
  2. Tap the "Start MR" switch: When MR is not active the switch is turned off, it displays the word Start MR and has a white background. When MR is active the switch is turned on, it displays the word Deactivate and has a yellow background. When MR is active and recording the switch is turned on, it displays the word Deactivate and has a purple background. MR settings cannot be accessed while active.
  3. Tap the "Current Settings" button: MR will display in a short message what are the current settings.
  4. Tap the "Close" button: MR will be closed and will be running in the background. MR will still be active if the switch is on, in this case the volume keys can still be used to record media even in cases that the device is locked (the user is notified about this through a short message).
  5. Tap the Help button: MR will display a short help section at the bottom part. The help section can be hidden by pressing the same button that displays "Hide Help".

Usually this can be done by opening MR again and granting the permissions it asks. Permissions can be granted through the device's settings at Settings->Applications->MR->Permissions. In some cases you may also need to go to Settings->Applications->MR->Advanced->Display over other apps in order to allow MR to work in the background.

All the recorded media is stored in the main storage of the device in the folder "MR". Inside the "MR" folder the user may find the following folders:

  • The folder "V": Here MR stores all the recorded video files that are not hidden.
  • The folder "A": Here MR stores all the recorded audio files that are not hidden.
  • The folder ".V": Here MR stores all the recorded video files that are hidden.
  • The folder ".A": Here MR stores all the recorded audio files that are hidden.

Recorded files with the option "Hide MR files" enabled are not shown in the gallery and could not be shown on your device if the device is not showing hidden files. The user has to enable the device's option to show hidden files.
Recorded files with the option "Hide MR files" disabled are shown in the gallery. It might take a while for new recorded files to appear in the gallery, they will appear after the device has updated the contents of the gallery.

Yes. MR will always react to the volume keys when it is active.

All MR files are named with the letters MR followed by a date and time. For instance, a video file could have the name MR_20190715_144115.mp4 and an audio file could have the name MR_20190715_144115.3gp. The extension .mp4 is used for video files and .3gp for audio files.

FAQ

Una vez MR es instalado y abierto puede pasar lo siguiente:

  1. MR pregunta por permisos: Dependiendo del dispositivo y de la versión de Android, MR puede preguntar por permisos para grabar audio, grabar video, guardar archivos en el dispositivo y/o permisos para ser mostrado sobre otras apps. Estos permisos son necesarios para que MR funcione y el usuario debe otorgar estos permisos a MR para poder grabar.
  2. MR muestra el mensaje "MR no tiene los permisos requeridos para grabar": En este caso MR necesita que el usuario le otorgue todos los permisos que MR necesita para que le sea posible grabar.
  3. MR es abierto y ningún mensaje es mostrado: En este caso MR está listo para funcionar.

Cuando MR está listo para ser usado, el usuario tiene las siguientes opciones:

  1. Ir a "Configuración" en los tres puntos verticales en la parte superior derecha: Aquí se puede modificar la configuración general y obtener información acerca de MR. En configuración general el usuario tiene las siguientes opciones:
    • Seleccionar la cámara de grabación de la lista de cámaras disponibles en el dispositivo.
    • Escoger la máxima longitud de grabación para cada archivo grabado.
    • Activar/desactivar notificaciones de vibración al inicio/final de cada grabación.
    • Escoger el modo de grabación entre archivos de audio o de video.
    • Activar/desactivar la opción de activación automática. Cuando la opción de Activación automática está activada MR será activado automáticamente cada vez que se abre la aplicación.
    • Activar/desactivar la opción de ocultar archivos. Cuando la opción de Ocultar archivos MR está activada todos los archivos grabados estarán ocultos de la galería del dispositivo.
  2. Presionar sobre el switch de "Activar MR": Cuando MR no está activo el switch estará apagado, mostrará las palabras Activar MR y tendrá un fondo color blanco. Cuando MR está activo el switch estará encendido, mostrará la palabra Desactivar y tendrá un fondo color amarillo. Cuando MR está activo y grabando el switch estará encendido, mostrará la palabra Desactivar y tendrá un fondo color purpura. El menú de configuración de MR no puede ser abierto mientras MR está activo.
  3. Presionar sobre el botón de "Configuración actual": MR mostrará la configuración actual por medio de un corto mensaje.
  4. Presionar sobre el botón de "Cerrar": MR se cerrará y estará en segundo plano. MR permanecerá activo si el switch está encendido, en este caso las teclas de volume aún podran ser usadas para grabar, incluso en casos en que el dispositivo está bloqueado (el usuario es notificado acerca de esto mediante un corto mensaje).
  5. Presionar sobre el boton de Ayuda: MR mostrará una corta sección de ayuda en la parte inferior. La sección de ayuda se puede cerrar presionando el mismo boton que mostrara las palabras "Ocultar ayuda".

Usualmente los permisos pueden ser otorgados en el momento en que MR es abierto y pregunta al usuario sobre los permisos que requiere. Para activar los permisos por medio de la configuración del dispositivo se debe ir a Configuración->Apps->MR->Permisos. En algunos casos el usuario puede necesitar ir a Configuración->Apps->MR->Avanzada->Mostrar sobre otras apps para permitir MR grabar en segundo plano.

Todos los archivos grabados son guardados en el almacenamiento principal del dispositivo en la carpeta "MR". Dentro de la carpeta "MR" el usuario puede encontrar las siguientes carpetas:

  • La carpeta "V": En esta carpeta MR guarda todos los archivos de video grabados que no están ocultos.
  • La carpeta "A": En esta carpeta MR guarda todos los archivos de audio grabados que no están ocultos.
  • La carpeta ".V": En esta carpeta MR guarda todos los archivos de video grabados que están ocultos.
  • La carpeta ".A": En esta carpeta MR guarda todos los archivos de audio grabados que están ocultos.

Archivos grabados con la opción de "Ocultar archivos MR" activa no son mostrados en la galería y puede que no aparezcan en el dispositivo si este no tiene la opción de mostrar archivos ocultos activada. El usuario debe activar en el dispositivo la opción de mostrar archivos ocultos.
Archivos grabados con la opción de "Ocultar archivos MR" desactivada serán mostrados en la galería. Puede tomar algunos instantes para que nuevos archivos grabados aparezcan en la galería, estos aparecerán una vez el dispositivo actualice la galería.
