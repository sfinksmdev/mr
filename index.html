Background Media Recorder

MR is a background media recorder that allows you to record video/audio without showing any preview on your device. The user can navigate through other apps while MR is recording. MR has the option to start/stop recordings by pressing the volume keys, even when the device is locked or the screen is off. You can select the source camera, set vibrating notifications, set the recording length, and use extra options on the paid version such as programming a timer or activate the reactive recording mode.

MR es una grabadora en segundo plano que le permite grabar video/audio sin mostrar ninguna vista preliminar. El usuario puede navegar en otras aplicaciones mientras MR está grabando. MR tiene la opción de empezar/terminar grabaciones con sólo presionar las teclas de volume, incluso cuando el dispositivo está bloqueado o la pantalla está apagada. La aplicación le permite seleccionar la cámara de grabación, configurar notificaciones de vibración, configurar la duración de grabación, y uso de opciones extras en la versión paga para programar un timer o activar el modo de grabación reactiva.
