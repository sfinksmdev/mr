MR is an application that allows you to record video/audio by pressing the volume keys without showing any preview. You can select the source camera, set vibrating notifications, and set the recording length. Once MR is active you can keep working on your phone and simply use the volume keys to instantly start/stop recording.

MR es una aplicación que permite grabar video/audio con sólo presionar las teclas de volume sin mostrar ninguna vista preliminar. El usuario puede seleccionar la cámara de grabación, configurar notificaciones de vibración y duración de grabación. Una vez MR está activo el usuario puede seguir usando el dispositivo y simplemente usar las teclas de volume para empezar/terminar grabaciones.


Main features

Record media with volume keys
Video recording
Audio recording
Vibrating notifications
Select recording camera from the cameras available
Autostart option when opened
Hide recorded files option
Permission to internet to check for updates and to load ads

Características principales

Grabación de contenido con las teclas de volumen
Grabación de video
Grabación de audio
Notificaciones con vibración
Selección de cámara de grabación
Opción de auto activación al abrir
Opción de ocultar archivos creados
Permiso para internet para verificar actualizaciones y cargar ads
